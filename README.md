# snake_game_bevy
Try to program the snake game by Bevy 0.12 <br>
Support touch screen swipe gesture <br>

[Keys]<br>
  directions: touch swipe / arrow keys / WSAD / KJHL<br>
  pause: space / ESC<br>
  confirm: space / return<br>
